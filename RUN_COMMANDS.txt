ALERT INCIDENT INTELLIGENCE - RUN COMMANDS

Project root:
/home/parth/alert-incident-intelligence

1) Activate Python environment
cd /home/parth/alert-incident-intelligence
source .venv/bin/activate

2) Start backend API (Terminal 1)
cd /home/parth/alert-incident-intelligence
source .venv/bin/activate
uvicorn backend.api:app --reload --host 0.0.0.0 --port 8000

3) Start frontend dashboard (Terminal 2)
cd /home/parth/alert-incident-intelligence/frontend
npm install
npm run dev

Frontend URL:
http://localhost:5173

Backend URL:
http://localhost:8000
API docs:
http://localhost:8000/docs

4) One-shot synthetic file ingest to backend (Terminal 3)
cd /home/parth/alert-incident-intelligence/synthesis
source /home/parth/alert-incident-intelligence/.venv/bin/activate
python3 ingest_client.py --source meraki --ingest-url http://localhost:8000/ingest
python3 ingest_client.py --source auvik --ingest-url http://localhost:8000/ingest
python3 ingest_client.py --source ncentral --ingest-url http://localhost:8000/ingest

5) Interval random synthesizer ingest (2 to 3 alerts per minute)
cd /home/parth/alert-incident-intelligence/synthesis
source /home/parth/alert-incident-intelligence/.venv/bin/activate
python3 ingest_client.py \
  --interval-seconds 60 \
  --cycles 20 \
  --sources meraki,auvik,ncentral \
  --random-synthesizer \
  --min-alerts-per-minute 2 \
  --max-alerts-per-minute 3 \
  --epochs 20 \
  --ingest-url http://localhost:8000/ingest

6) Train model from database data
cd /home/parth/alert-incident-intelligence
source .venv/bin/activate
python model/training_the_model.py --use-db --window-minutes 10 --max-dt-minutes 240 --log-level DEBUG

7) Optional: train model from CSV instead of DB
cd /home/parth/alert-incident-intelligence
source .venv/bin/activate
python model/training_the_model.py --input stitched_alerts_dedup.csv --window-minutes 10 --max-dt-minutes 240 --log-level DEBUG

8) Quick API checks
curl http://localhost:8000/health
curl "http://localhost:8000/alerts?limit=20"
curl "http://localhost:8000/alerts/ml?limit=20"

9) Open PostgreSQL database
psql -h localhost -p 5432 -U parth -d alerts_db

Inside psql:
\dt
